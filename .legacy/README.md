# 레거시 자료

이 폴더는 문서의 목적·구조·전제·본문 대부분을 다시 써야 현재 코드의 계약으로 사용할 수 있는 과거 작업 문서를 보관합니다.

- 현재 구현 계약은 루트 `README.md`, `docs/`, `SESSION-HANDOFF.md`를 우선합니다.
- 이 폴더의 문서는 과거 결정 배경과 작업 순서를 확인하는 참고 기록입니다.
- 레거시 문서의 파일 경로, 수치, 미구현 목록을 새 작업의 기준으로 사용하지 않습니다.
- 일부 문구·경로·수치만 고치면 정합 가능한 문서는 이 폴더로 옮기지 않고 `docs/`에서 현행화합니다.
- 현재 보관 문서: `docs/implementation-plan.md` — 완료된 클릭 액션 구현 계획으로, 현행 문서로 사용하려면 전면 재작성해야 합니다.
